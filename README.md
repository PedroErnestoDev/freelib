# Projeto Freelib

O projeto **Freelib** é um portfólio que é dividido em duas pastas principais: **frontend** e **backend**. Nessa aplicação, utilizamos uma variedade de tecnologias para garantir uma experiência completa e moderna.

## Tecnologias Utilizadas
- **ReactJS**
- **PHP**
- **JavaScript**
- **Sass**
- **MySQL**
 

Neste projeto, o frontend é responsável pela interface do usuário, enquanto o backend se encarrega da lógica de servidor e manipulação de dados.
